What is PDS?

PDS, or Package Distribution Search tool, helps you to search for available packages across OS distributions in a single user interface. Current version of PDS is v1.3

Which operating systems does PDS support?

Currently, PDS supports the following operating systems:

  • SUSE Linux Enterprise Server 11 SP4/12 SP1/12 SP2
  • Ubuntu 16.04/16.10/17.04

Where can I find more details about packages available for various distros?

More details of packages can be obtained directly from following links provided by respective distros.

  • SUSE Linux Enterprise Server 11
  • SUSE Linux Enterprise Server 12
  • Ubuntu 16.04
  • Ubuntu 16.10
  • Ubuntu 17.04

Does PDS list the packages for all architectures?

No. The packages that PDS lists are for only the s390x architecture.

Which web browsers does PDS work on?

Currently, PDS works on the following web browsers:

  • Mozilla Firefox 45.7.0
  • Google Chrome 55
  • Internet Explorer 11

What is the format for entering the search keywords?

The keyword must be three or more characters.

  • To search for a package that contains the keyword anywhere in its name, enter three or more characters
  • To search for a package that contains the keyword at the end of its name, enter the asterisk character, followed by three or more characters. For example, enter *docker to search for package names that end with the string docker.
  • To search for a package that contains the keyword at the beginning of its name, enter three or more characters followed by the asterisk character. For example, enter docker* to search for package names that begin with the string docker.
  • To search for package names that exactly match the keyword, enter three or more characters, and then click the Search (Exact Match) hyperlink.

How can I narrow my search?

To get packages which exactly matches the search keyword, click on the hyperlink "Search <keyword> (Exact Match)" after entering search keyword

What package details does PDS display in the search results?

PDS displays the following package details in the search results:

  • Package Name
  • Package Version
  • Operating system for which the package is available

How accurate is the package information that PDS provides?

The package information is regularly updated. It is accurate as on the date of the update. However, the information might become obsolete between updates. To obtain the most current package information, go to the official Linux distribution websites.
